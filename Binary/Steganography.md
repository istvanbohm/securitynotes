
# Strings

Display printable strings in a file

`strings`

# Binwalk

Search binary files (e.g. images,audio files) for embedded files and data

## Display data

`binwalk`

## Extract data

`binwalk -e file`

# Steghide

Hide or extract data

## Display info

`steghide info file.jpg`

## Extract data

`steghide extract -sf file.jpg`
